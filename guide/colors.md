# 主题色

系统内置的主题色可自由定义扩展。

## ThemeColorArray

在 `@/store/modules/SysStore` 中有 `ThemeColorArray` 变量。

```typescript
/**
 * 系统内置的主题颜色数组
 * https://material.io/resources/color/#!/?view.left=0&view.right=0
 * 取值为600的色值
 */
const ThemeColorArray = [
  "#039be5",
  "#e53935",
  "#d81b60",
  "#8e24aa",
  "#5e35b1",
  "#3949ab",
  "#1e88e5",
  "#00acc1",
  "#00897b",
  "#43a047",
  "#7cb342",
  "#c0ca33",
  "#fdd835",
  "#ffb300",
  "#fb8c00",
  "#f4511e",
  "#6d4c41",
  "#757575",
  "#546e7a",
];
```
